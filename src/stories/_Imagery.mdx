# Imagery

<div className="subheading">Iconography</div>
Most icons used in Modderize are from the [Hero Icon](https://heroicons.com/) set
(usually solid or outline variants). These are imported directly into the code as
React components, but can be downloaded as SVGs from the Hero Icons website for use
in mockups and prototypes. [![Image from Gyazo](https://i.gyazo.com/8d50240c72bc8fb5da4ecc22aaea59ac.gif)](https://gyazo.com/8d50240c72bc8fb5da4ecc22aaea59ac)

The other icons are typically brand icons, which are usually from the [react-share package](https://www.npmjs.com/package/react-share).
[![Image from Gyazo](https://i.gyazo.com/d27d98772e59f0198c1c45494904ee5e.png)](https://gyazo.com/d27d98772e59f0198c1c45494904ee5e)

<div className="subheading">Photography</div>
Photography used throughout the website is all sourced from royalty free media sites.
[Unsplash](https://www.unsplash.com/) and [Pexels](https://www.pexels.com/) are the
two I've used the most. Static photos should be compressed after downloading and
uploaded in webp format.

<div className="subheading">User-Uploaded Images</div>
User-uploaded images such as screenshots for mods or avatars are typically uploaded
as JPEGs; these are less performant, but are a more common format for lay users than
webp. There are hard limits on file size when it comes to these images, although
those differ by the recommended resolution. On average, keeping user upload file
sizes beneath 500kb is the goal.
