# Typography

The selected font for the Modderize platform is [Inter](https://fonts.google.com/specimen/Inter), a free sans-serif font created by Google.
[![Image from Gyazo](https://i.gyazo.com/d16930b4c236552919a426a99354b177.gif)](https://gyazo.com/d16930b4c236552919a426a99354b177)

The base font for all text in the repo is this one at 16 pixels and 400 weight. See the "body text" section for an example of what this looks like.

<div className="subheading">Links and Buttons</div>
The same font-family is used for links and buttons, as well, albeit in different
colors and with slightly different styling.

For detailed information on how links are displayed, see the [A component documentation](/docs/modderize-a--docs).

<div className="subheading">Jumbotron</div>
Similarly to links and buttons, the text in Jumbotrons shares similarities and differences
with the largest headers.

For detailed information on how jumbotron headings are displayed, see the [Jumbotron component documentation](/docs/modderiz-a--docs).
